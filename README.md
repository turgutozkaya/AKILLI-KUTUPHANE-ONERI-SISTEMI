#  Smart Library System

##  Features
- **User & Book Management:** Güvenli veritabanı işlemleri (CRUD).
- **Reading History:** Kullanıcıların okuma geçmişi ve puanlama sistemi.
- **Book Search:** `LIKE` operatörü ile detaylı kitap arama.
- **Smart Recommendation System:** Kullanıcının geçmiş okumalarını analiz edip (GROUP BY) sadece **okumadığı** kitapları öneren gelişmiş algoritma.
- **User Profile Analysis:** Kullanıcının okuma alışkanlıklarını yüzdelik (%70 Bilim Kurgu vb.) olarak çıkaran analiz motoru.
- **Robust Architecture:** Modüler dosya yapısı ve `try-except` bloklarıyla çökmeye karşı korumalı sistem.

##  Technologies
- Python (Modular Structure)
- SQLite (Relational Database)
